Biology
My powerpoint, presented to the class to inform them of the effects of mutations in the gene for von Willebrand Factor.

Transformation is seen here in that when the gene is mutated, it changes the effect and function of proteins.
von Willebrand Disease is used as the example in this project. Upon normal function, it assists in the clotting of blood. Reduction of this could cause slower clotting, or a lack of clotting, and result in prolonged bleeding from minor cuts, heavy bruising, large nose bleeds, etc.
If a mutation occured in the von Willebrand gene when it was assembled, the gene would code for different amino acids, the building blocks of proteins. Because proteins are three dimentional, and the shape is due to the interactions between the amino acids, changing one would cause the shape to alter. This could effect the function of the protein, causing it to either not be as efficient, not work at all, or even possibly work better. It could also not effect the function at all, this is possible if the amino acid that was changed is of the same type as was intended, causing no change in the shape of the protein.
